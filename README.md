# Android AdBlocker App

A simple yet effective Android application to block ads system-wide by intercepting DNS traffic using the Android `VpnService` API and routing it to AdGuard DNS.

## 📁 Repository Structure
- **[AdBlockApp/](./AdBlockApp/)**: Contains the source code for the Android application.
- **[AdBlockApp/README.md](./AdBlockApp/README.md)**: Detailed documentation and user guide in **Arabic**.

## 🚀 Quick Start
To build the application, ensure you have the Android SDK installed and run:
```bash
cd AdBlockApp
gradle assembleDebug
```
The APK will be generated at `AdBlockApp/app/build/outputs/apk/debug/app-debug.apk`.
