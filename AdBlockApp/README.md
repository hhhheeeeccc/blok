# مانع الإعلانات (AdBlocker)

تطبيق أندرويد بسيط يقوم بمنع الإعلانات في التطبيقات والمواقع عن طريق استخدام خدمة VPN محلية تقوم بتوجيه طلبات DNS إلى خوادم AdGuard DNS المصفاة.

## المميزات
- واجهة بسيطة باللغة العربية.
- زر لتفعيل ومنع الإعلانات.
- زر لإيقاف الخدمة.
- لا يحتاج إلى صلاحيات الروت (Root).

## كيف يعمل
يستخدم التطبيق فئة `VpnService` في أندرويد لإنشاء نفق VPN محلي. بدلاً من تشفير كل البيانات، يقوم فقط بتغيير خوادم DNS للجهاز إلى:
- 94.140.14.14
- 94.140.15.15

هذه الخوادم تمنع الوصول إلى نطاقات (Domains) الإعلانات والتعقب تلقائياً.

## المتطلبات
- أندرويد 7.0 (API 24) أو أحدث.

## البناء (Build)
يمكنك بناء التطبيق باستخدام Gradle:
```bash
gradle assembleDebug
```
سيظهر ملف الـ APK في المسار:
`app/build/outputs/apk/debug/app-debug.apk`
