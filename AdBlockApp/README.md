# مانع الإعلانات (AdBlocker)

تطبيق أندرويد متطور وبسيط مصمم لمنع الإعلانات المزعجة التي تظهر فجأة في التطبيقات والمواقع، مما يوفر تجربة تصفح أسرع وأكثر أماناً.

## المميزات الرئيسية
*   **حماية شاملة**: يمنع الإعلانات في المتصفحات وداخل التطبيقات الأخرى.
*   **واجهة عربية عصرية**: تصميم سهل الاستخدام يدعم اللغة العربية بالكامل مع وضع ليلي (Dark Mode).
*   **بدون روت (No Root)**: يعمل التطبيق على جميع أجهزة أندرويد دون الحاجة لصلاحيات النظام المتقدمة.
*   **خصوصية عالية**: لا يقوم التطبيق بتتبع نشاطك، بل يكتفي بتصفية طلبات DNS.

## كيف يعمل التطبيق؟
يعتمد التطبيق على تقنية **VpnService** المدمجة في أندرويد لإنشاء "نفق" محلي للبيانات. يقوم هذا النفق بتوجيه طلبات العناوين (DNS) فقط إلى خوادم **AdGuard DNS** المتخصصة.
هذه الخوادم تمتلك قائمة سوداء محدثة باستمرار لعناوين الإعلانات والتعقب، وتقوم بحجبها تلقائياً قبل أن تصل إلى جهازك.

## المكونات التقنية
*   **AdBlockVpnService**: الفئة المسؤولة عن إدارة اتصال الـ VPN وتغيير إعدادات الـ DNS للجهاز.
*   **MainActivity**: واجهة المستخدم التي تمكنك من التحكم في حالة الحماية (تفعيل/إيقاف).

## خوادم DNS المستخدمة
يستخدم التطبيق الخوادم التالية لضمان أفضل حماية:
*   Primary: `94.140.14.14`
*   Secondary: `94.140.15.15`

## متطلبات التشغيل
*   نظام أندرويد 7.0 (API Level 24) أو أحدث.

## طريقة بناء التطبيق (Build)
للمطورين، يمكن بناء ملف الـ APK باستخدام Gradle عبر الأمر التالي:
```bash
cd AdBlockApp
gradle assembleDebug
```
بعد الانتهاء، ستجد الملف الناتج في:
`app/build/outputs/apk/debug/app-debug.apk`
